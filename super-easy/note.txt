# üî• Top 10 Must-Memorize Coding Patterns (FAANG)

---

## 1Ô∏è‚É£ Linear Scan (Single Pass)

**When to use:**
Find min, max, sum, count, condition match.

**Template idea:**

```js
for (let i = 0; i < arr.length; i++) {
  // update answer
}
```

**Used in:**
Max element, sum of array, count vowels
**Time:** O(n)

---

## 2Ô∏è‚É£ Two Pointers

**When to use:**
Reversal, palindrome, sorted array problems.

**Template idea:**

```js
let left = 0, right = arr.length - 1;
while (left < right) {
  left++;
  right--;
}
```

**Used in:**
Reverse string, palindrome check
**Time:** O(n)

---

## 3Ô∏è‚É£ Sliding Window

**When to use:**
Subarray / substring of fixed or variable size.

**Template idea:**

```js
let left = 0;
for (let right = 0; right < n; right++) {
  // expand window
  // shrink window if needed
}
```

**Used in:**
Max sum subarray, longest substring
**Time:** O(n)

---

## 4Ô∏è‚É£ Frequency Map (Hash Map)

**When to use:**
Counting, duplicates, matching patterns.

**Template idea:**

```js
const map = {};
map[x] = (map[x] || 0) + 1;
```

**Used in:**
Anagrams, first non-repeating char
**Time:** O(n)

---

## 5Ô∏è‚É£ Hash Set (Existence Check)

**When to use:**
Duplicate detection, fast lookup.

**Template idea:**

```js
const set = new Set();
if (set.has(x)) {}
set.add(x);
```

**Used in:**
Remove duplicates, intersection
**Time:** O(n)

---

## 6Ô∏è‚É£ Prefix Sum

**When to use:**
Range sum queries, subarray sum problems.

**Template idea:**

```js
prefix[i] = prefix[i - 1] + arr[i];
```

**Used in:**
Subarray sum equals K
**Time:** O(n)

---

## 7Ô∏è‚É£ Sorting + Scan

**When to use:**
Order matters or easier grouping.

**Template idea:**

```js
arr.sort((a, b) => a - b);
```

**Used in:**
Anagrams, merge intervals
**Time:** O(n log n)

---

## 8Ô∏è‚É£ Binary Search

**When to use:**
Sorted data, monotonic condition.

**Template idea:**

```js
while (low <= high) {
  const mid = Math.floor((low + high) / 2);
}
```

**Used in:**
Search element, first/last position
**Time:** O(log n)

---

## 9Ô∏è‚É£ Recursion (Basic)

**When to use:**
Repeating sub-problems, tree traversal.

**Template idea:**

```js
function dfs(x) {
  if (base) return;
  dfs(next);
}
```

**Used in:**
Factorial, Fibonacci, tree DFS
**Time:** Depends

---

## üîü Stack Pattern

**When to use:**
Previous/next element, parentheses problems.

**Template idea:**

```js
const stack = [];
stack.push(x);
stack.pop();
```

**Used in:**
Valid parentheses, next greater element
**Time:** O(n)
